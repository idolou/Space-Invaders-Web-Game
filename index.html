    About Space Invaders Game

    Creators: Ido Lourie and Elad Shmulevich

    This Web game called Space Invaders, your goal is to kill all bad spaceships!

    Game Keys:

      Up: use the ⬆ Key

      Down: use the ⬇ Key

      Left: use the ⬅ Key

      Right: use the ➡ Key

      Shot: Space Bar by deafult, you can choose your own Key!


    We Add the following functionality:

    1.The option to choose the player spaceship.

    2. The option to choose the shotkey for the game

    3. The option to choose time for the game by time slider

    3. Game Timer That indicates how much time left


    The challenges we had are:

    2. Validate the sing up form and check all fields

    3. Reset The Game
